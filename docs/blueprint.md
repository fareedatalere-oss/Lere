# **App Name**: Lere Connect

## Core Features:

- Secure User Registration & Login: Allow users to create an account with phone number, username, password, transaction PIN, and an optional referral code. Provide secure login and session management.
- Personalized Dashboard Display: Show the user's phone number, account number, current balance, and prominent buttons for 'Voice Call' and 'Video Call'.
- One-to-One Video Call: Users can initiate and receive real-time video calls with other registered users within the application.
- One-to-One Voice Call: Users can initiate and receive real-time voice calls with other registered users.
- Simplified Financial Actions: Include interactive buttons for 'Buy Data', 'Buy Airtime', and 'Send Money' which navigate to basic input forms for demonstration purposes in an MVP.
- AI Call Ice-Breaker Tool: An AI tool to generate a fun, engaging question or topic suggestion to help users initiate conversations before a call.
- User Profile Management: Allow users to view and update their personal details including username, phone number, and password.

## Style Guidelines:

- Light color scheme to convey clarity and a modern, friendly aesthetic. The primary color is a medium, communicative blue (#226EC9) which anchors trust and technology. The background color is a heavily desaturated blue (#ECF2F7) to maintain visual continuity while providing a clean canvas. An analogous accent color, a vibrant cyan (#31DCDC), provides contrast and highlights interactive elements.
- All text will use 'Inter', a grotesque-style sans-serif font. Its modern, objective, and neutral aesthetic ensures excellent readability and a sleek appearance for all headlines and body text, aligning with the app's focus on clear communication.
- Utilize modern, clean outlined icons for dashboard elements, call actions, and financial features, maintaining visual simplicity and immediate recognition.
- A clean, mobile-first layout prioritizing ease of access to core communication features. The dashboard will have distinct sections for user account information, large call buttons, and neatly arranged financial action buttons.
- Implement subtle, fluid animations for navigation transitions, button feedback upon tap, and smooth visual cues when initiating or receiving a call, enhancing user experience without distraction.
